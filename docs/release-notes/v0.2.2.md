# DockerForge v0.2.2

DockerForge generates and lints production-grade Dockerfiles from the command line. This release focuses on stronger npm and GitHub presentation, release hygiene, and digest pinning.

## Highlights

- Added opt-in Docker Hub digest pinning with `dockerforge generate --pin-digests`.
- Kept default generation offline unless digest pinning is requested.
- Added clearer Docker build, run, and Compose instructions.
- Tidied public docs so internal planning notes are not part of the public tree.
- Added contributor guidance and GitHub issue and pull request templates.

## Packages

- `@dockerforge/cli`: install with `npm install -g @dockerforge/cli`.
- `@dockerforge/core`: install with `npm install @dockerforge/core`.

## Quick Start

```bash
npx @dockerforge/cli generate ./my-app
docker build -t my-app ./my-app
docker run --rm -p 3000:3000 my-app
```

For digest-pinned Docker Hub base images:

```bash
npx @dockerforge/cli generate ./my-app --pin-digests
```

## Verification

Before publishing, run:

```bash
npm run verify
```
