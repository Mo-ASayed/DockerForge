# DockerForge v0.2.5

This patch release fixes service detection when projects contain Terraform cache directories.

## Highlights

- DockerForge now ignores `.terraform` directories during service discovery.
- Prevents cached Terraform modules from being detected as deployable Go services.
- Keeps the real root app selected in frontend repos that also contain Terraform state/cache files.
